# Snake Game
A simple snake game created using SDL2. 

The purpose of this project is to recreate the classic Snake Game using C++'s SDL Library.

# Controls

| Key  | Action |
| --- | --- |
| UP  | Move snake uo  |
| DOWN  | Move snake down  |
| LEFT  | Move snake left  |
| RIGHT  | Move snake right  |
| SPACE  | Restart game  |
| ESCAPE  | Close game  |
